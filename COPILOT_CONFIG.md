# 繁體中文 Copilot 配置指南\n\n## 📋 配置文件總覽\n\n本項目已為 VS Code Copilot 配置繁體中文支持。以下文件控制 Copilot 的行為：\n\n| 文件 | 用途 | 適用範圍 |\n|------|------|----------|\n| [.instructions.md](./.instructions.md) | 通用指令規範 | 所有 Copilot 互動 |\n| [.prompt.md](./.prompt.md) | Chat 提示和範例 | Copilot Chat |\n| [.rules.md](./.rules.md) | 項目規則 | 代碼質量和風格 |\n\n---\n\n## 🎯 關鍵特性\n\n### ✅ 一律繁體中文\n- 所有技術說明使用繁體中文\n- 代碼保持英文（便於編程）\n- 一致的中文術語對應\n\n### ✅ 智能代碼建議\n- 理解項目結構和規範\n- 提供符合項目風格的代碼\n- 自動驗證代碼質量\n\n### ✅ 完整的項目上下文\n- QC-Web IPQC 項目特性\n- API 設計規範\n- 數據庫約定\n\n---\n\n## 🚀 如何使用\n\n### 1. 確認 Copilot 已識別配置\n打開任何代碼文件，Copilot Chat 應該自動應用這些設置。\n\n### 2. 在 Chat 中使用\n```\n問題：「怎樣優化這個查詢？\"\n回應：Copilot 將用繁體中文回答\n```\n\n### 3. 查看配置生效\n- Copilot 回應使用繁體中文 ✅\n- 代碼建議遵循項目規範 ✅\n- 術語一致使用 ✅\n\n---\n\n## 📖 配置文件詳解\n\n### .instructions.md - 通用指令\n\n**包含內容**:\n- 語言設定（一律繁體中文）\n- 代碼規範（縮排、路徑等）\n- 回應格式要求\n- 項目特性說明\n\n**使用場景**:\n- 代碼生成\n- 代碼解釋\n- 代碼審查\n- 文件操作\n\n### .prompt.md - Chat 提示\n\n**包含內容**:\n- 回應風格指南\n- 常見使用情景的範例\n- 技術詞彙對應表\n- 對話範例（好的/不好的）\n\n**使用場景**:\n- Copilot Chat 對話\n- 技術討論\n- 問題排查\n- 功能實現建議\n\n### .rules.md - 項目規則\n\n**包含內容**:\n- 語言規則\n- 代碼質量標準\n- 文件組織結構\n- 技術規範（前/後端）\n- API 設計規則\n- 安全和性能要求\n\n**使用場景**:\n- 代碼質量檢查\n- 新功能設計\n- 代碼審查\n- 性能優化\n\n---\n\n## 💡 常見問題\n\n### Q: 為什麼 Copilot 回應還是用英文？\n\n**A**: \n1. 確認文件已保存到項目根目錄\n2. 重新打開 VS Code 或工作區\n3. 在 Copilot Chat 中明確要求：「請用繁體中文回答\"\n4. 檢查 Copilot 版本是否為最新\n\n### Q: 代碼中應該用繁體中文嗎？\n\n**A**: \n✅ **應該**用繁體中文的地方:\n- 代碼註解\n- 字符串中的用戶消息\n- 文檔和 README\n\n❌ **不應該**用繁體中文的地方:\n- 變數名\n- 函數名\n- 類名\n- 引用的英文庫名\n\n### Q: 如何覆蓋配置？\n\n**A**: \n在 Chat 中明確說明，例如：\n```\n\"使用英文回答這個問題\"\n```\nCopilot 會遵循您的即時指示。\n\n### Q: 配置會影響代碼完成嗎？\n\n**A**: \n不會。這些配置只影響：\n- Copilot Chat 的回應\n- 代碼建議的說明\n- Inline completions 的文本\n\n不影響代碼片段本身。\n\n---\n\n## 🔧 自訂配置\n\n### 修改語言設定\n\n如果需要改為其他語言，編輯 `.instructions.md` 和 `.prompt.md` 中的語言相關部分。\n\n### 添加項目特定規則\n\n在 `.rules.md` 中的適當部分添加新規則，格式：\n```markdown\n✅ 必須遵守：\n- 規則1\n- 規則2\n```\n\n### 調整技術術語\n\n在 `.prompt.md` 的術語表中更新或添加術語。\n\n---\n\n## 📊 配置優先級\n\n當 Copilot 收到指令時，優先級如下（從高到低）：\n\n1. **用戶的即時指示** (Chat 中明確說明)\n2. **.prompt.md** (Chat 特定指導)\n3. **.instructions.md** (通用指令)\n4. **.rules.md** (項目規則)\n5. **Copilot 默認值**\n\n---\n\n## ✅ 驗證檢查清單\n\n確認配置正確應用：\n\n- [ ] 所有三個文件都存在於項目根目錄\n- [ ] 文件使用 UTF-8 編碼\n- [ ] 文件名以點開頭 (`.instructions.md` 等)\n- [ ] 打開 VS Code 後 Copilot 顯示繁體中文回應\n- [ ] 代碼建議遵循項目規範\n- [ ] Chat 能理解項目上下文\n\n---\n\n## 📝 最佳實踐\n\n### 1. 清晰的提問\n```\n❌ 不好：「修改這段代碼\"\n✅ 好的：「添加錯誤處理並用繁體中文返回錯誤消息\"\n```\n\n### 2. 提供上下文\n```\n❌ 不好：「怎樣優化？\"\n✅ 好的：「這個 API 端點每秒收到 1000 個請求，查詢需要優化\"\n```\n\n### 3. 明確目標\n```\n❌ 不好：「檢查代碼\"\n✅ 好的：\"檢查是否有 SQL 注入漏洞和性能問題\"\n```\n\n---\n\n## 🔗 相關資源\n\n### VS Code Copilot 文檔\n- [Copilot 官方指南](https://github.com/github/copilot-docs)\n- [自訂指令說明](https://github.com/github/copilot-docs/blob/main/docs/using-copilot-chat-in-vs-code.md)\n\n### 項目文檔\n- [QUICK_REFERENCE.md](./.claude/QUICK_REFERENCE.md) -
